Professional Writing Class Assignments

Ollie Instructions

For my professional writing class, I was assigned to make instructions. I choose to write about how to ollie, since I skateboard often and have spent a lot of time learning and improving at the trick. In this document, I communicated my ideas clearly, which is a useful skill for working with a team and brainstorming. I also was able to communicate complex processes, which I did mainly by breaking the task down into manageable steps such as the one shown below.

When writing these steps I focused on the user, which is an important skill for when designing software that needs to be beginner or user friendly. One of the ways I did this was through writing common mistakes as shown below, many of which I made when first learning how to ollie.

These common mistakes sections also show my attention to detail in combating problems the reader might face. This is similar to debugging and optimizing code where it helps to cover potential problems in the future and think about corner cases.
